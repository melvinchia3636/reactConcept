## Dicectory

### Components

- Variable Component
  ```jsx
  const Component = <></>;
  ```
- Functional Component
  ```jsx
  function Component() {
    return <></>;
  }
  ```
- Arrow Function Component
  ```jsx
  const Component = () => {
    return <></>;
  }
- Class Component
  ```jsx
  class Component extends React.Component {
    render() {
      return <></>;
    }
  }
  ```
